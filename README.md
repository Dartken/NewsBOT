Бот с новостями по теме

Короткое описание:
Бот, который каждый день присылает тебе свежие и важные новости по выбранной теме — без лишнего шума.

Расширенное описание:
Этот бот помогает быть в курсе того, что действительно важно именно тебе. Просто задай тему (например: “искусственный интеллект”, “криптовалюта”, “маркетинг” или даже узкую нишу вроде “стартапы в Европе”), и бот будет ежедневно отправлять тебе краткую выжимку актуальных новостей.

Основной функционал:
1. Выбор темы
Пользователь вводит:
одно слово (“AI”)
несколько тем (“финансы, стартапы”)
или даже сложный запрос (“регулирование криптовалют в ЕС”)

2. Ежедневная рассылка
Бот:
собирает новости (через API / парсинг)
фильтрует по теме
делает краткое резюме (через ChatGPT)
отправляет 3–10 ключевых новостей

3. Формат сообщения
Каждая новость:
Заголовок
Краткое описание (2–4 предложения)
Ссылка на источник
Почему это важно (опционально)

4. Настройки пользователя
Пользователь может:
изменить тему
выбрать время получения
выбрать язык
задать количество новостей (например 3 / 5 / 10)

5. Умная фильтрация
удаление повторов
отсев нерелевантных новостей
приоритизация важных событий
